Important note: By default, SMART is loading a complete stylesheet file that contains all the styles needed for the entire template. However, if you'd like to reduce bloat and load a minimal stylesheet with the required styles on each page of your application, follow along.


Usage

  1. Create a new LESS file (e.g. styles.less) and save it in the assets/less folder.

  2. Copy & paste the following imports in the LESS file you just created, in this order:

    CORE minimal styles:
    @import "../assets/library/bootstrap/css/bootstrap.min.css";
@import "../assets/library/icons/fontawesome/assets/css/font-awesome.min.css";
@import "../assets/library/icons/glyphicons/assets/css/glyphicons_regular.css";
@import "../assets/library/icons/glyphicons/assets/css/glyphicons_social.css";
@import "../assets/library/icons/glyphicons/assets/css/glyphicons_filetypes.css";
@import "../assets/components/core/variables.less";
@import "../assets/components/core/mixins.less";
@import "../assets/components/core/scaffolding.less";
@import "../assets/components/core/helpers.less";
    Component styles:
    @import "../assets/components/ui_pagination/pagination.less";
  3. Place the file in your project's document root.
  4. Load the LESS file into the <head> section of your HTML document, before any JavaScript files: 

    <link rel="stylesheet/less" href="styles.less" />

    NOTE  To include the LESS file directly, make sure the LESS.js library is also loaded in the head section of the template, otherwise compile the LESS file into CSS and include that file.

  5. If you'd like to include other components on the same page, don't create multiple LESS files, but add all the imports in one file. You can create a LESS file for each page with just the resources used for that specific page, or you could create and use a single LESS file for the entire project.
  6. In production, compile the LESS file and use the resulting minified CSS file in the HTML document.
